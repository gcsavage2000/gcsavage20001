The problem I set out to solve in Lab 6 was how to select specific areas that aren't already neatly divided into enumeration units.

Specifically, I evaluated: What kinds of problems call for geo-processing operations instead of selections? How do you get the math of a Union to work for our problem solving areas? And, why are buffers and geographic distance useful tools for selecting (or not selecting) specific areas? The skills I learned through this Lab and making this map include how to use the following tools: Buffers, Multipart to Singlepart, Dissolve, Spatial Join, Clip, Erase, Intersect, and Union.

To get my solution, I started by creating an inclusive layer of all the areas where the new store could be located. This meant buffering the regional rail stations and subway stations, using the union tool to join the resulting shapefiles together, and then using the intersect tool to intersect this layer with the empowerment zones. Then I created an exclusive layer where the store could not be located by buffering the farmers markets and corner stores, and using the union tool on the resulting layers.

I then created the candidate areas by erasing the exclusive areas from the inclusive areas, and used the multipart to singlepart tool to clean up the candidates layer. The problem we were trying to solve in this lab and in creating this map was how to make choropleth and proportional symbol maps using Census data at a national scale. Specifically, this lab aimed to answer the questions: Where does Census data come from? How is it collected? What projection do you use when mapping the continental US? What are the limitations of Census data? How do you choose the proper map type for your data and audience?

This map is a choropleth of the percentage of the Hispanic population by state in the contiguous US states.The skills I learned in this lab include how to use map layout, symbology, elements, placements, and data field identification and use for the purpose of proportional symbol and choropleth maps.To solve this lab’s problem and create this map I first used select by attribute to select only the contiguous US states and created a new shapefile of these states. To make the choropleth of the percentage of the Hispanic population by state, I navigated to the symbology pane, changed the symbology type to graduated colors, and selected the Hispanic population variable as the field. Then under normalization, I selected the total population variable, and selected quantile as the method. I then adjusted the color scheme and histogram break lines, and created a new layout and added this map to it. Lastly, I added the relevant title and graphic elements, and used the share tab to export my maps as a PNG file.
